# Block Shooter Game

A simple yet engaging WebGL-based space shooter game where you control a spacecraft and defend against falling blocks.

#Gameplay Mechanics

1. Your Mission
   - Control a spacecraft at the bottom of the screen
   - Shoot and destroy falling blocks before they reach the bottom
   - Survive as long as possible while scoring points

2. Scoring System
   - Each destroyed block awards 10 points
   - Try to achieve the highest score possible!

3.*Block Types
   - Blocks have different health points (1-3 hits to destroy)
   - Stronger blocks require multiple hits to destroy
 
4. Game Over Conditions
   - Collision with a block
   - Block reaches the bottom of the screen

Tips & Strategies
- Keep moving to avoid falling blocks
- You can hold down the spacebar for continuous firing
- Plan your movements ahead as blocks fall at different speeds
